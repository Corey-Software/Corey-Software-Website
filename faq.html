  • What can a resource pack change in Minecraft Bedrock Edition

    Resource packs can change pretty much everything you see or hear in Minecraft. They can change the designs (textures) applied to blocks, mobs, entities, etc. They can change the shape of mobs and entities to be something different. They can change the music and sounds that the game makes to be different sounds or to be played differently. They can change the actual text that you see on screen to something different, such as changing all occurrences of Roman numerals to Arabic numerals (IV -> 4).

  • What "complicated stuff" does Resourceful handle

    The main thing that Resourceful does is it takes the 2D textures that Minecraft uses and makes them 3D so that you can edit a texture while it looks like it does in the game instead of editing it while it is flat. Beyond that, Resourceful also handles the naming of files, file structure, and some JSON editing. As new updates come out it will be able to do more to make the resource packs you create with Resourceful be more feature-rich.

  • Why make a tool for Minecraft Bedrock Edition instead of Minecraft Java Edition

    There are a couple of reasons to make Resourceful for Minecraft Bedrock Edition. The first is that I primarily play on Bedrock Edition. This means I have the most experience with the platform that I am developing for.

    The second is that there are already a lot of tutorials for how to make resource packs for Minecraft Java Edition, but not very many for Bedrock Edition. This means that Resourceful would be more important for the Bedrock community since it replaces the need for tutorials.

    The third is that Bedrock Edition is on more platforms. Java is on Windows, Mac, and Linux, whereas Bedrock is on Windows, Consoles (Xbox, Playstation, Switch), and Mobile (Apple, Android). This means that the resource packs you create can be shared with your friends*.

    The final reason is the environment. From my experience, people on Java Edition tend to play on servers when they want to play with other people. Usually, servers have their own resource packs that they want their players to use, thus diminishing the importance of custom resource packs. Also, Bedrock Edition players tend to join a friend’s world/realm when they want to play together. This lends itself to using and sharing a custom resource pack since it is more informal and less regulated than a typical server.

    *Currently it is difficult to transfer resource packs to consoles (Xbox, Playstation, Switch). However, they can still easily experience resource packs hosted on other people's worlds.

  • Will Resourceful ever be able to make resource packs for Java Edition too

    Probably. Bedrock Edition is a great place for me to start, but that doesn't mean I want to just do Bedrock Edition. I think it is a good idea to expand Resourceful’s compatibility to Java Edition.

  • What platforms are Resourceful being developed for

    Currently, Resourceful is being developed to run on Windows 10. Other platforms such as macOS and Linux may be added if the demand is high enough. Also, if it could work well, a tablet version for iPadOS and Android may also be a possibility.

  • Does Minecraft allow custom resource packs

    Yes! Minecraft fully supports resource packs whether they come from the Minecraft Marketplace built into the game or from somewhere else like Resourceful or a download from a friend. Minecraft Bedrock Edition makes it super easy to add custom resource packs to your game and to share packs with your friends due to Bedrock Edition’s .mcpack filetype. Also, unlike behavior packs (modpacks), resource packs don’t disable achievements, meaning that it isn’t considered cheating by the game.

  • Will Resourceful be able to make RTX resource packs

    Making custom RTX resource packs sounds really cool and I would love to be able to do that, but there are a couple of problems with it. The first is the size of the audience. RTX resource packs require an Nvidia RTX graphics card on Windows 10. That is a very small segment of the Minecraft community. The second problem is that it is very different from normal resource packs meaning it would be a lot of time and effort on my part to get it working. I think it would be a better use of my time to spend that time and effort working on features to benefit all platforms rather than just Windows 10 computers with Nvidia RTX graphics cards.

  • What’s the difference between Java and Bedrock resource packs

    There are a couple of differences that I have noticed so far (I don’t do much with Java resource packs). The main difference is that Java Edition has blockstates JSON files that allow for some pretty cool things including connected textures. Other than that, there are some differences in the file structure of the resource packs as well as some differences in the names of the textures. All of the differences mean that you cannot use a Java Edition resource pack on Bedrock Edition and vice versa.

  • If it’s so hard to learn how to make a resource pack, how did you do it

    I learned the basics of Bedrock Edition resource packs from the sparse tutorials online and learned the more advanced things by downloading the vanilla (default) Bedrock Edition resource pack from Minecraft’s site and tinkering with things to see what changed. Also, while programming Resourceful, whenever something looks different in Resourceful than it does in Minecraft, I find out why instead of just fixing it.
